# Image Placeholder Guide

This directory should contain the following images for the website:

## Logo
- `thaiboranlogo.png` (height: 60px recommended)
  - Logo image for header and footer
  - Use PNG with transparent background

## Homepage Background
- `thaiboranwall.jpg` (1920x600px recommended)
  - Background image for hero section

## Thai Boran Standard Section
- `thaiboranmassager.jpg` (400x400px recommended)
  - Image for the Thai Boran Standard section
  - Should show a massage therapist or person receiving massage

## Service Images
Place your service images in this folder:

- `hotstonemassage.jpg` (250x250px recommended)
  - Image for Hot Stone Massage service
  - Should show hot stone massage treatment

- `aromatherapy.jpg` (250x250px recommended)  
  - Image for Aroma Therapy service
  - Should show essential oils/aromatherapy

- `scalpmassage.jpg` (250x250px recommended)
  - Image for Scalp Massage service
  - Should show scalp massage/head treatment

## How to Add Images

1. Prepare your images (250x250px works well for service cards)
2. Name them exactly as listed above
3. Place them in this folder: `/public/images/`
4. The React components will automatically load them

## Image Format
- Format: JPG, PNG, or WebP
- Recommended size: 250x250px for service cards, 1920x600px for hero, 60px height for logo
- File size: Keep under 200KB for optimal loading

## Example Paths
```
/public/images/thaiboranlogo.png
/public/images/thaiboranwall.jpg
/public/images/thaiboranmassager.jpg
/public/images/hotstonemassage.jpg
/public/images/aromatherapy.jpg
/public/images/scalpmassage.jpg
```

Once images are added, they will appear automatically on the website!
